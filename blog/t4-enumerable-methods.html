Enumerable Method #Method

11.23.2014

So, first thing's first. What is an enumerable? Admittedly, I found myself confused every time I came across the term.

Enumerable: a toolbox of handy methods to use with classes like arrays and hashes A class is eligible to use enumerables if it can also use the #each method. The #each method is used to iterate through a block of code, like this:

salutations = ["hi", "greetings", "wassup"]
salutations.each do |words|
puts words + " Sarah"
end
The result:
hi Sarah
greetings Sarah
wassup Sarah

Let's examine the #map method.
#map is synonymous with the #collect method (but is more commonly used).
Use this to return a new array with the results from iterating through the block.

map { |obj| block } => array
array = [1,2,4,8]
array.map { |x| x * 2 } #=> [2, 4, 8, 16]
