The 4 Ways To Position Elements

November 9th, 2014

  1. Static
    2. position: static
    All elements start off as static but it is rarely utilized.
    Usage: Rare

  2. Relative
    3. "Relative to itself"
    position: relative
    The element is positioned in relation to where it would normally be.
    To use, make sure to include another positioning attribute to shift the element (top, bottom, left, right).
    Usage: Common but often misused
    Ex: position: relative left: 20px
  3. Absolute
    4. position: absolute
    The attributes you set will be relative to the parent element with relative or absolute positioning.
    If there is no parent, the element will place itself relative to the page.
    Absolute positoning is not affected by other elements.
    Pros: Place an element anywhere you choose!
    Cons: Overusing this type of positioning limits the flexibility of the website.
    Usage: Common, easily overused
  4. Fixed
    5. position: fixed
    Element is positioned relative to the browser window and stays put when page scrolls.
    Pro: Creates a cool effect because element is always present
    Con: Usability/can cut off content
    Usage: Rare
